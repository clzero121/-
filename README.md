# 유클리드 호제법

두 개의 자연수의 최대공배수를 구하는 알고리즘

## 과정
1. a를 b로 나눈다. (a > b)
2. 1.에서 나온 b는 a가 되고 나머지는 b가 된다.
3. 나머지가 0이 될때까지 1-2를 반복한다.
4. 최대공약수는 나머지 0이 되기 전 단계의 나머지이다.
```
ex) 2484 4212

4212 = 2484 × 1 + 1728

2484 = 1728 × 1 + 756

1728 = 756 × 2 + 216

756 = 216 × 3 + '108' << 최대공약수

216 = 108 × 2 + 0
```
